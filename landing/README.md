# GEMS Landing Page

Landing page gioi thieu trai nghiem doc cua GEMS.

## Deploy tren Vercel

1. Import repository `hoanganhthai2104/gems-ebook`.
2. Dat **Root Directory** thanh `landing`.
3. Chon **Framework Preset** la `Other`.
4. De trong Build Command va Output Directory.
5. Deploy.

Trang la HTML tinh, khong can cai dependency hoac chay build.
